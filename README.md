# guide-wap-jiuyou

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文件形式存放，不绑定任何特定域名或网站。

## 项目简介

这是一个内容归档性质的仓库，主要用途是保存和公开分享一些独立的 HTML 页面。每个页面在仓库中均以独立文件或文件夹形式存在，便于直接访问或引用。

## 目录说明

```
.
├── README.md          # 本说明文件
├── index.html         # 仓库主页（可选）
├── page-xxx/          # 具体页面文件夹（示例）
│   └── index.html
└── ...
```

- 根目录下可放置全局入口页面（如 `index.html`）。
- 每个独立页面通常放置于一个独立文件夹中，文件夹名称与页面内容相关。
- 所有页面均为纯静态 HTML，不依赖后端服务。

## 页面归档说明

- 每个归档页面均为独立完整的 HTML 文件，可单独打开使用。
- 页面内容可能涵盖工具、说明、演示等不同用途，具体请参考每个页面内的介绍。
- 页面资源（如图片、样式、脚本）尽量内联或随文件夹附带，以保证离线可用性。

## 维护说明

- 本仓库不定期更新，新增页面或修正已有页面。
- 若发现页面无法正常打开或内容有误，欢迎提交 Issue 反馈。
- 不接受针对具体网站或商业用途的修改请求。

## 使用方式

- 直接通过 GitHub Pages 或其他静态托管服务访问仓库中的 HTML 文件。
- 也可克隆仓库到本地，在浏览器中打开对应页面。
